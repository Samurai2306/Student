# Анализатор сетевых подключений (WPF)

учебное WPF‑приложение на .NET 8, которое показывает информацию о сетевых интерфейсах компьютера и позволяет разбирать и проверять URL/URI.

## Что сделано

- Показывает список всех сетевых интерфейсов (Ethernet, Wi‑Fi, VPN и т.д.).  
- Для выбранного интерфейса отображает:
  - имя, описание, тип и состояние;
  - скорость соединения;
  - основной IPv4‑адрес и маску подсети;
  - MAC‑адрес;
  - все IP‑адреса интерфейса (IPv4/IPv6);
  - шлюзы, DNS‑серверы, MTU и простую статистику трафика.
- Разбирает введённый URL/URI: схема, хост, порт, путь, параметры, фрагмент.
- Проверяет доступность хоста через **Ping** (время ответа, TTL).
- Получает **DNS‑информацию** (список IP и обратное имя).
- Определяет тип адреса: loopback, локальный (частный), публичный, link‑local IPv6.
- Хранит историю проверенных URL (можно выбрать из списка и переанализировать).

## Как запустить

Требуется установленный **.NET SDK 8**.

1. Перейти в папку проекта:

   ```bash
   cd P_2/NetworkAnalyzer
   ```

2. Собрать и запустить:

   ```bash
   dotnet run
   ```

Откроется окно приложения:

- слева — список интерфейсов и информация о выбранном;
- справа — поле для ввода URL и результаты анализа;
- снизу — история проверенных URL.
