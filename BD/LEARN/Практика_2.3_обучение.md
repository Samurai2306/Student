# Практика №2.3 — обучение (PL/pgSQL: IF, FOR, DIAGNOSTICS)

Ниже — ровно тот синтаксис PL/pgSQL, который нужен для понимания отчёта по практике 2.3.

## 1) Процедурные конструкции PL/pgSQL

PL/pgSQL позволяет писать функции с логикой (циклы, условия, переменные). Включается так:

```sql
LANGUAGE plpgsql
```

Тело функции оформляется как блок:

```sql
AS $$
DECLARE
  -- переменные
BEGIN
  -- код
END;
$$;
```

## 2) Условия `IF / ELSIF / ELSE`

Используется для выбора ветки:

```sql
IF cond THEN
  ...
ELSIF other_cond THEN
  ...
ELSE
  ...
END IF;
```

## 3) Цикл `FOR rec IN SELECT ... LOOP`

Частый паттерн «пройтись по строкам запроса»:

```sql
DECLARE
  rec RECORD;
BEGIN
  FOR rec IN
    SELECT ...
  LOOP
    -- rec.column_name
  END LOOP;
END;
```

- `RECORD` — «строка неизвестной структуры», удобно для произвольного `SELECT`.
- Внутри цикла поля доступны как `rec.address`, `rec.diff` и т.п.

## 4) Возврат набора строк из PL/pgSQL: `RETURNS TABLE` + `RETURN NEXT`

Если функция возвращает много строк:

```sql
RETURNS TABLE(col1 TEXT, col2 NUMERIC)
```

Внутри присваиваешь значения выходным колонкам (они становятся «переменными»), затем:

```sql
col1 := rec.address;
col2 := rec.diff;
RETURN NEXT;
```

## 5) CTE `WITH` и обновления `UPDATE ... FROM ...`

### 5.1) CTE (`WITH`)

`WITH` позволяет структурировать запрос:

```sql
WITH a AS (...),
b AS (...)
UPDATE ...
FROM b
WHERE ...;
```

### 5.2) `UPDATE ... FROM`

PostgreSQL умеет обновлять таблицу, используя данные из другой таблицы/CTE:

```sql
UPDATE properties p
SET price_rub = ...
FROM to_update u
WHERE p.id = u.id;
```

## 6) Среднее значение и условия скидок

### 6.1) Средняя оценка

```sql
AVG(r.rating)
```

и группировка:

```sql
GROUP BY p.id, ...
```

### 6.2) Дата «старше N месяцев»

В отчёте используется сравнение с интервалом:

```sql
created_date <= (CURRENT_DATE - INTERVAL '6 months')
```

## 7) Фильтрация агрегатов: `HAVING`

Если условие использует `SUM/AVG/...`, применяют `HAVING`:

```sql
GROUP BY ...
HAVING ABS(total_area - SUM(room_area)) > 0.01
```

## 8) Абсолютное значение и округление

- `ABS(x)` — модуль.
- `ROUND(x, 2)` — округление до 2 знаков после запятой (полезно для денег).

## 9) Сколько строк затронул `UPDATE`: `GET DIAGNOSTICS ... = ROW_COUNT`

После `UPDATE` можно получить число изменённых строк:

```sql
GET DIAGNOSTICS updated_count = ROW_COUNT;
```

`updated_count` — обычная переменная, которую ты объявляешь в `DECLARE`.
