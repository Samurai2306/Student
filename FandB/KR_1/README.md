# Фронтенд и бэкенд разработка — Практические занятия 1–6

Репозиторий содержит выполнение практических заданий по дисциплине «Фронтенд и бэкенд разработка» (4 семестр, 2025/2026). Каждое занятие размещено в отдельной папке.

## Содержание

| Папка | Тема | Описание |
|-------|------|----------|
| **practice_01** | CSS-препроцессоры (SASS) | Карточка товара: переменные ($primary-color, $spacing и др.), миксин card-base, вложенность селекторов |
| **practice_02** | Node.js + Express | REST API с CRUD для товаров (id, название, стоимость), валидация cost, логирование запросов |
| **practice_03** | JSON и внешние API | Отчёт по тестированию API в Postman и работе с внешним API |
| **practice_04** | API + React | Интернет-магазин: клиент на React, сервер на Express (≥10 товаров, карточка: название, категория, описание, цена, кол-во, рейтинг, фото); валидация на сервере, вывод ошибок API в интерфейсе |
| **practice_05** | Расширенный REST API | Подключение Swagger (swagger-jsdoc, swagger-ui-express), JSDoc для схемы Product и всех CRUD, документация на /api-docs |
| **practice_06** | Подготовка к КР №1 | Инструкции по тестированию и оформлению отчёта |

## Быстрый старт

### Практика 1
Открыть в браузере `practice_01/index.html`. CSS уже скомпилирован; при изменении SASS: `npx sass practice_01/styles.scss practice_01/styles.css`.

### Практика 2
```bash
cd practice_02
npm install
npm start
```
API: http://localhost:3000 (маршруты: `/products`, `/products/:id`).

### Практика 4 (интернет-магазин)
**Сервер:**
```bash
cd practice_04/server
npm install
npm start
```
**Клиент (в другом терминале):**
```bash
cd practice_04/client
npm install
npm run dev
```
Открыть http://localhost:3001

### Практика 5 (с Swagger)
```bash
cd practice_05/server
npm install
npm start
```
- API: http://localhost:3000  
- Документация: http://localhost:3000/api-docs  

Клиент — как в Практике 4 (подключается к тому же API).

## Требования

- Node.js (рекомендуется LTS)
- Для ПЗ 1: при необходимости — Sass (`npm install -g sass`)
